# Security policy

Report security-sensitive problems privately to the repository owner instead of placing secrets or exploit details in public issues.

LLVS feedback must be minimal and sanitized. Never attach API keys, tokens, passwords, private design files, customer screenshots or production data. The bundled collector redacts common credential patterns but is not a substitute for human review.

The public repository does not authorize access to third-party Figma files, paid features, private repositories or consuming-project assets.
